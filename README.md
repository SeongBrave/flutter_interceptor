# flutter_interceptor

flutter dio interceptor

## Getting Started


```dart
dependencies:
  flutter_interceptor: ^0.0.1
//Initialization
_dio.interceptors.add(UiNetInterceptor())
Overlay.of(context)?.insert(InterceptorDraggable());
```
